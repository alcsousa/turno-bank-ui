# TurnoBank API

## Project Setup

```shell
cp .env.example .env
```

Review `.env` values

```dotenv
VITE_API_BASE_URL=http://localhost
```

Install dependencies
```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
